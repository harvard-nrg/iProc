# External Dependencies

These are the software packages iProc was developed with and validated 
against.

!!! question "Can I upgrade these dependencies?"

    iProc may run perfectly fine with newer versions of the dependencies 
    listed below, but the results have never been validated. In other 
    words, YMMV.

Software                       | Versions
-------------------------------|------------------------------------
[Python][Python]               | `3.11`
[GNU Parallel][GNU Parallel]   | `20180522`
[MRIcron][MRIcron]             | `2012_12`
[AFNI][AFNI]                   | `2016_09_04`
[FSL][FSL]                     | `4.0.3`, `5.0.4`, `5.0.10`, `6.0.1`
[MRIcroGL][MRIcroGL]           | `2019_09_04`
[FreeSurfer][FreeSurfer]       | `6.0.0`
[ImageMagick][ImageMagick]     | `6.7.8-10`
[ANTs][ANTs]                   | `2.4.4`
[dcm2niix][dcm2niix]           | `1.0.20230411`
